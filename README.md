# RubricatePHP kickoff
Folder suggestion

## Installation
```
$ git clone https://github.com/rubricate/kickoff.git projectname
$ cd projectname
$ composer install
```
or:
## Installation 2:
```
$ composer create-project rubricate/kickoff projectname 
```
## In the browser: 
```
localhost/projectname/public
```
doc: https://rubricate.github.io/components/kickoff/
